# KRISHI-DRISHTI - Comprehensive Commit History

## Project Overview
KRISHI-DRISHTI (Farmer's Vision) is an AI-based crop disease detection system designed for rural farmers. The system enables farmers to capture leaf images using smartphones and receive instant disease diagnosis in Hindi along with treatment advice. The application is designed to work offline, ensuring accessibility in areas with limited connectivity, while helping reduce crop loss and improving agricultural productivity and decision-making.
