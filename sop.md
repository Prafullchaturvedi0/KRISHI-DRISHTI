# Statement of Purpose

This project, Krishi Drishti (Farmer’s Vision), is an AI-powered crop disease detection and advisory system designed to assist farmers in identifying plant health issues using image-based analysis. The system leverages machine learning and computer vision techniques to provide accurate, real-time diagnoses along with actionable treatment recommendations. It is built as a practical application of artificial intelligence aimed at solving real-world agricultural challenges, particularly for smallholder farmers.

## Problem statement

Farmers, especially in rural regions, often face difficulty in accurately identifying crop diseases at an early stage due to limited access to agricultural experts. Misdiagnosis or delayed treatment can lead to reduced yield and financial losses. Additionally, there is a lack of simple, technology-driven tools that are both accessible and easy to use for non-technical users.

The goal of this project is to develop a system that enables farmers to upload images of plant leaves and receive immediate, reliable disease identification along with suggested remedies. At the same time, the project serves as a practical demonstration of applying machine learning concepts such as image classification in a real-world scenario.

## Scope of the project

- Provide an interface (web or application-based) for uploading plant leaf images.
- Use a trained machine learning model to classify and detect crop diseases.
- Display disease name along with treatment or preventive measures.
- Ensure accessibility by designing a simple and intuitive user interface.
- Support real-time or near real-time predictions.
- Demonstrate integration of AI models with a functional application.
- Serve as an educational project showcasing the use of:
  - Computer vision techniques
  - Machine learning model deployment
  - Data preprocessing and prediction pipelines
- Limitations:
  - Accuracy depends on training data quality and variety.
  - Limited to specific crops/diseases included in the dataset.
  - Requires image input of sufficient quality.
  - May not function effectively without internet (if deployed online).
  
## Target users

- Farmers seeking quick and reliable crop disease identification.
- Agricultural students and researchers exploring AI applications in farming.
- Developers and learners interested in machine learning and computer vision projects.
- Organizations working toward digital agriculture and smart farming solutions.

## High-level features

- Image-based crop disease detection using machine learning.
- User-friendly interface for uploading and analyzing images.
- Instant prediction results with disease name and possible remedies.
- Efficient backend model for fast processing and response.
- Scalable architecture that can be extended to include:
  - More crops and diseases
  - Multilingual support
  - Offline functionality
  - Integration with agricultural advisory systems
- Demonstrates key technical concepts:
  - Image preprocessing and classification
  - Model training and inference
  - Application integration with AI systems

By implementing this project, the aim is to bridge the gap between advanced AI technologies and practical agricultural needs, while also providing a strong learning foundation in machine learning, software development, and real-world problem solving.
